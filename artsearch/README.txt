This is the README.txt file for Art Search Exhibit.


###   How to run:

1) Make sure you have the following packages installed:
    bs4
    csv
    itertools
    nltk
    numpy
    os
    pandas
    pickle
    re
    requests
    scipy

2) Change working directory to ccr/artsearch.

3) Run server in terminal by typing the following:
    python manage.py runserver
    
4) Go to local host in web browser:
    localhost:8000
